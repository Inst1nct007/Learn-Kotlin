5. WAP to print the number of digits in a number

Soln.

    val num: Int = 12
    
    val digits: String = when (num) {
        in 1..9 -> "1"
        in 10..99 -> "2"
        in 100..1000 -> "3"
        else -> "More than 3 digits"
    }
    
    print(digits)
